Comandos de Linux

Primeros pasos

Aprendiendo a caminar en la terminal

Comando Descripción
ls Lista todas las carpetas y archivos del directorio actual
ls -l Lista carpetas y archivos con detalles
ls -d Lista solo carpetas del directorio actual
ls -lh Lista carpetas y archivos con detalles y tamaño en MB
cd <directorio> Cambia de directorio, sin parámetros lleva a home
cd .. Cambia de directorio al padre
clear Limpiar la consola, tambien se puede limpiar con Ctrl + L
pwd Muestra el directorio actual
file <archivo> Muestra el tipo de archivo

Manipulando archivos y directorios

Comando Descripción
ls -la Lista carpetas y archivos ocultos
ls -lr Lista carpetas y archivos en orden inverso
tree Muestra la estructura de directorios
tree -L <nivel> Muestra la estructura de directorios hasta el nivel especificado
mkdir <nombre> Crea un directorio con el nombre especificado
touch <nombre> Crea un archivo con el nombre especificado
cp <origen> <destino> Copia un archivo o directorio a otro lugar
mv <origen> <destino> Mueve un archivo o directorio a otro lugar
rm <archivo> Elimina un archivo, -i para confirmar la eliminación
rmdir <directorio> Elimina un directorio si está vacío
rm -r <directorio> Elimina un directorio y su contenido

Explorando el contenido de nuestros archivos

Comando Descripción
head <archivo> Muestra las primeras líneas de un archivo
head -n <número> <archivo> Muestra las primeras líneas de un archivo con el número de líneas especificado
tail <archivo> Muestra las últimas líneas de un archivo
tail -n <número> <archivo> Muestra las últimas líneas de un archivo con el número de líneas especificado
less <archivo> Muestra el contenido de un archivo con paginación, / para buscar palabras
xdg-open <archivo> Abre un archivo con el programa predeterminado, Ctrl + C para salir

Qué es un comando

Comando Descripción
type <comando> Muestra el tipo de comando
alias <nombre>="comando" Crea un alias para un comando
unalias <nombre> Elimina un alias
help <comando> Muestra la ayuda de un comando
man <comando> Muestra el manual de un comando
whatis <comando> Muestra la definición de un comando
apropos <palabra> Muestra los comandos que contienen la palabra especificada

Wildcards

Comando Descripción
* Muestra todas la coincidencias con cualquier secuencia de caracteres
? Muestra todas la coincidencias con un solo carácter
[[:upper:]] Muestra todas la coincidencias con cualquier carácter en mayúscula
[[:lower:]] Muestra todas la coincidencias con cualquier carácter en minúscula
[letras] Muestra todas la coincidencias con cualquier carácter en la lista de letras
[[:alnum:]] Muestra todas la coincidencias con cualquier carácter alfanumérico
[[:alpha:]] Muestra todas la coincidencias con cualquier carácter alfabético
[[:digit:]] Muestra todas la coincidencias con cualquier carácter numérico

Empezando a correr

Redirecciones: Cómo funciona la shell

Comando Descripción
> Redirecciona la salida de un comando a un archivo
>> Redirecciona la salida de un comando a un archivo, concatena el contenido del archvio
2> Redirecciona la salida de un comando a un archivo de error
<comando> > <archivo> 2>&1 Redirecciona la salida de un comando a un archivo y la salida de error a otro archivo

Redirecciones: pipe operator

Comando Descripción
echo <mensaje> Muestra un mensaje en la consola
cat <archivo> Muestra el contenido de un archivo
cat <archivo1> <archivo2> Concatena el contenido de dos archivos
<comando> | <comando> Redirecciona la salida de un comando a la entrada de otro comando
tee <archivo> Crea un archivo con la salida de un comando
sort Ordena el contenido de un archivo

Encadenando comandos: operadores de control

Comando Descripción
<comando> ; <comando> Ejecuta dos comandos secuencialmente, síncrona
<comando> & <comando> Ejecuta dos comandos en segundo plano, asíncrona
<comando> && <comando> Ejecuta dos comando de forma condicional, si ambos son verdaderos
<comando> || <comando> Ejecuta dos comando de forma condicional, si uno de ellos es verdadero
